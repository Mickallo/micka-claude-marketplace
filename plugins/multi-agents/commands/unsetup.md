# Subagent Unsetup

Remove the delegation rules previously injected by `/setup` from the project's `CLAUDE.md`.

## Steps

1. **Read the project CLAUDE.md** — read `CLAUDE.md` at the project root. If it doesn't exist, inform the user and stop.

2. **Find the section** — locate the `## Delegation` section (identified by the `<!-- Generated by /setup — do not edit manually -->` comment)

3. **Remove it** — delete the entire `## Delegation` section (from the heading to the next `##` heading or end of file)

4. **Clean up** — remove any trailing blank lines left behind

5. **Report** — confirm to the user that the delegation rules have been removed.
