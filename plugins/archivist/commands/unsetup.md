# Archivist Unsetup

Remove archivist behavior rules from the project's `CLAUDE.md`.

## Steps

1. **Read the project CLAUDE.md** — read `CLAUDE.md` at the project root. If it doesn't exist, report that there's nothing to remove.

2. **Find the `## Archivist` section** — locate the section starting with `## Archivist` and ending before the next `## ` heading (or end of file).

3. **Remove it** — delete the entire section including the heading.

4. **Write back** — save the updated CLAUDE.md.

5. **Report** — confirm to the user that the archivist rules were removed.
